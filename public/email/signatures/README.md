## Setup Instructions

### Outlook Desktop (Windows)

1. Open the signature HTML file in a browser
2. Select all (`Ctrl+A`) and copy (`Ctrl+C`)
3. In Outlook: **File → Options → Mail → Signatures → New**
4. Name it (e.g. "IE - James") and paste (`Ctrl+V`) into the editor
5. Set as default for your info@ or sponsors@ account
6. Click OK

### Outlook Desktop (Mac)

1. Open the signature HTML file in a browser
2. Select all (`Cmd+A`) and copy (`Cmd+C`)
3. In Outlook: **Outlook → Settings → Signatures → +**
4. Name it and paste into the editor
5. Assign to the correct account

### Outlook Web (OWA)

1. Go to **Settings (gear icon) → Mail → Compose and reply**
2. Under "Email signature", click **+ New signature**
3. Name it (e.g. "IE - Mike")
4. Open the HTML file in a browser, select all, copy, and paste into the signature box
5. Set as default signature for new messages and/or replies
6. Save

### Apple Mail (macOS)

1. Open the HTML file in Safari
2. Select all (`Cmd+A`) and copy (`Cmd+C`)
3. In Apple Mail: **Mail → Settings → Signatures**
4. Click `+` to add a new signature
5. Paste (`Cmd+V`) into the signature editor
6. Drag the signature to the correct email account in the left panel
7. Uncheck "Always match my default message font" at the bottom

### Apple Mail (iOS / iPadOS)

1. iOS doesn't support rich HTML signatures natively
2. Best option: set up the signature in Outlook Web (OWA) — it carries across to the Outlook mobile app
3. Alternatively, email yourself the HTML file, open it in Safari, copy, and paste into **Settings → Mail → Signature**. Formatting may be limited.
